    A aplicação aceita o nome do ficheiro YAF a
ler por argumento ($ yaf.exe T01G04_solarsystem.xml).
    Ao compilar, todos os ficheiros na pasta resources
(texturas e .yaf) são copiados para a mesma pasta onde
é colocado o yaf.exe (/bin).
    Para compilar o projeto em Visual Studio 2010 é
necessário utilizar a configuração "Debug_v100". Para
compilar em Visual Studio 2012 usa-se a configuração
"Debug_v110" e a configuração "Debug" destina-se a
Visual Studio 2013.
